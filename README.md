## 🛠️ FITUR UTAMA (ADVANCED OPTIMIZATION)

Script ini melakukan serangkaian tweak sistem yang dirancang khusus untuk perangkat MIUI guna menghilangkan lag dan meningkatkan responsivitas:

* **MIUI Native Tweaks**: Mengaktifkan strategi performa baru dan profil daya tinggi (`IMQSNative`).
* **Foreground Focus**: Memprioritaskan sumber daya CPU/RAM untuk aplikasi yang sedang dibuka.
* **FPS Unlocker**: Memaksa sistem berjalan di **144Hz** dan mengaktifkan `fixed-performance-mode`.
* **Game Compilation**: Mengompilasi paket Free Fire & Free Fire Max dengan mode `speed` untuk loading lebih cepat.
* **Visual Optimization**: Mempercepat skala animasi sistem menjadi 0.5x.
* **Custom Resolution**: Mengubah resolusi ke **1280x2850** dengan density **400 DPI** untuk pandangan yang lebih luas.
